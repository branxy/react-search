## Запуск

1. Клонировать проект (или скачать ZIP):

```
git clone https://github.com/branxy/react-search.git
```

2. Перейти в проект:

```
cd react-search
```

3. Установить зависимости:

```
npm i
```

4. Запуск:

```
npm run dev
```
